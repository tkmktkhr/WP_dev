Required is a simple, clean theme for the modern blogger.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Theme Dashboard
2. Select `required.zip` from your computer
3. Upload
4. Activate the plugin on the WordPress Theme Dashboard

= Using FTP =

1. Extract `required.zip` to your computer
2. Upload the `required` directory to your `wp-content/themes` directory
3. Activate the theme on the WordPress Plugins dashboard
